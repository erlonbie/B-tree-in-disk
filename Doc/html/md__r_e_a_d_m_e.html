Como rodar o projeto

Rodar em 1 comando:

make run-all

Rodar o container:

make run

Alocar espaço para o hash:

make hash-allocate

Compilar os arquivos cpp:

make compile-upload
make compile-findrec
make compile-seek1

To end:

make end
